# Flash-Card

Website to give your learning to a effective mean.


## Tech Stack
### Frontend:
* HTML
* CSS
* Boostrap
* Javascript

### Backend+db (preferred):
* Flask
* SQLite

## Components

### Login and Register
User have to register and login first to access the web.

### Create Deck
User can create a deck and add cards into the deck.

### Update and Delete Deck
User can delete and update decks.

## Ongoing work :man_technologist:
Creating better login and logout system and working on ORM for the database.
